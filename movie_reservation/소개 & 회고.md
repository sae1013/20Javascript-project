### 회고
그동안 프레임워크에 절여있었는데, 바닐라 js로 DOM api 사용하는 연습이 되었다.
- 이벤트 위임
```js
$container.addEventListener("click", (e) => {
  if (
    e.target.classList.contains("seat") &&
    !e.target.classList.contains("occupied")
  ) {
    e.target.classList.toggle('selected');
    updateSelectedCount();
  }
});
```
